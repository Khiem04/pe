Variable and data types

Things to remember about variable and data types

#1: Python naming rules

- Must start with a letter and separated with underscore(if the variable has 2 character)

- Variables are case-sensitives(eg. age, Age and aGe are 3 different variables)

More about the rules

Data type

- Text type: string(str)

- Numeric type: integer(int), float , complex

- Sequence type: list, tuple, range

- Mapping type: dictionary(dict)

- Set types : Set, frozenset

- Boolean types: Boolean(bool)

- Binary types: bytes,bytearray, memoryview
