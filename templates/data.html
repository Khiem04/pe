Stack data structure

Content summary:

- Definition: A structure that store item in the "Last In/First Out" manner and happened on both side of the stack.

- Stack operators: append(), pop(), empty(), size(), top().

- Application: Memory management, backtracking(sudoku)

Queue data structure

Content summary:

- Definition: A linear structure that store items in the" First In- First out" manner.

- Queue operators: Enqueue(add an item into the queue), Dequeue(remove an item from a queue), Front(Get the front item of the queue), Rear(Get the last item of the queue)

- Application: Ordering food items,CPU task scheduling, Time reminder, etc

Linked list data structure

Content summary:

- Definition: A sequence of data elements, which are connected together through links. Each element will contain a connection to a element

- Different between linked list and array:

+ Array is a collection of similar data while Linked list contains a collection of unordered linked elements as nodes

+ The element in array belong to their indexes, meaning that we can take any number from the array by just enter the variable name and its position, while Linked list need to go from the beginning

+ Element are stored consecutively in arrays while it stored randomly in Linked lists.

- Application: Previous and next page in web browser, music player,etc

Graph data structure

Content summary:

- Definition: An non-linear structure including nodes and edges. While nodes are points on the graph, edge are the line connected each nodes together

- Algorithms use in Graph: Depth-first search(DFS) and Breadth-first search(BFS)

+ Depth-first search(DFS): An edge-based technique, using Stack data structure to perform. The algorithm starts from the root node and go along the branch before the backtracking.

+ Breadth-first search(BFS): An node-based technique, using Queue data structure to perform. The algorithm started by choosing an nodes and continue to move to another layer when first layer are finished.

+ Application: Finding shortest path, GPS navigation, etc.
