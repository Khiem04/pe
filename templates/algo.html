Bubble sort

Easiest sorting algorithms

#1: Definition of bubble sort algorithm

- An sorting algorithms that compared each of the value in a list and repeated swapping until they are in the correct order.

How does it work?

First run

- (6,4,1,2,5)=> (4,6,1,2,5) as the algorithm recognize that 6>4

- (4,6,1,2,5)=> (4,1,6,2,5) as the algorithm recognize that 6>1

- (4,1,6,2,5)=> (4,1,2,6,5) as the algorithm recognize that 6>2

- (4,1,2,6,5)=> (4,1,2,5,6) as the algorithm recognize that 6>5

END OF FIRST RUN
Second run

- (4,1,2,5,6)=> (1,4,2,5,6) as the algorithm recognize that 4>1

- (1,4,2,5,6)=> (1,2,4,5,6) as the algorithm recognize that 4>2

- (1,2,4,5,6)=> (1,2,4,5,6) as the algorithm recognize that 4<5 --> the list has been ordered in correct way

Third run

Even though the list has been ordered in the correct way, the algorithm need to identify the whole list has been ordered in the correct way, therefore this run will run but without any changes

Merge sort

Another sorting algorithms

#1: Definition of merge sort algorithm

- An algorithm that follow the "Divide and conquer" method, meaning that the array will be divide into two halves, then merge the two sorted halves

How does it work?

Step 1: Divide the the array into two parts by finding the middle element

Step 2: Divide the array into two parts again

Step 3: Break each element into single parts

Step 4: Sort the element from smallest to largest

Step 5: Merge the two sorted arrays

Quick sort

Another sorting algorithms

#1: Definition of quick sort algorithm

- An algorithm that follow the "Divide and conquer" method by choosing a pivot element
- The "pivot" element can be the first, last, random or the median of the array

How does it work?

Step 1: Choosing the pivot element( Partition)

Step 2: Rearrange the array so that the value that is smaller than the pivot element go before, and all value that is greater than the pivot element go after(equal values can go either way). After this, the pivot is in its final position

Step 3: Repeat step above with sub-array element with smaller value and separate it with greater values
